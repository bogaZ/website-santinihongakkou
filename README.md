## How to install
1. `composer install`
2. `cp .env.example .env`
3. create database and edit file .env
4. `php artisan migrate`
5. `php artisan db:seed`
6. `rm -rf public/storage`
7. `php artisan storage:link`
8. `npm install`
9. `npm run build`
10. `node link-node-modules.js`
11. `php artisan serve`
12. `npm run dev`
13. access website
